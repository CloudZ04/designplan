## Design Plan

![status](https://img.shields.io/badge/-CURRENTLY%20IN%20DEVELOPMENT-a04000?style=for-the-badge)

A visual design planning tool built with Next.js.

## Features

- Design plan dashboard
- Canvas workspace
- Drag and organise design components
- PostgreSQL storage
